Highlighted Projects

Rancid Tomatillos is a Turing School of Software and Design paired project. The purpose of this project is to create a dynamic application that leverages an API to receive, and display, movie data. A user can search and filter a long list of movies, as well as view detailed information about a specific movie by clicking on it. To create this application, we leveraged React, React Router, JavaScript, HTML, and CSS to create the front-end. In addition, we used Cypress for test-driven development of our application.
